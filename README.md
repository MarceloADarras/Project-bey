# Proyecto Bey

Proyecto personal de aplicación para amantes del beyblade, que cuenta con usos tanto informativos en los mismos beyblades, como de interacción entre usuarios, tanto para conocerse como para vender productos asociados 
al beyblade. 

Proyecto aún en desarrollo.

---

## [Backend](/backend/README.md)

- Python (3.11.9)
- Django (4.1)
- MySql

---

## [Frontend](/frontend/README.md)

- Node (22.19.0)
- npm (10.9.3)
- Vue.js (3.5.26)
- TailWind (3.4.19)
